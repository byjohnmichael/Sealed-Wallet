# Sealed-Wallet
Sealed Wallet is a set of python scripts to enable you to create and manage air-gapped crypto wallets

### Wallets in development (Listed in order of priority)
Active Development:
- Bitcoin
- Ethereum
- Solana
- Stellar
- XRP

To be considered:
- Cardano
- Bitcoin Cash
- Ethereum Classic
- Litecoin
- Avalanche
- Tezos
- Dogecoin
